Detrital MC user guide
======================

.. figure:: https://github.com/HUGG/Detrital-MC/raw/master/img/BZ435_ECDFs.png
   :width: 800 px
   :alt: Detrital MC ECDF plot

   Predicted and observed detrital thermochronometer age distributions plotted as empirical cumulative distribution functions

Detrital MC is a Monte Carlo code for comparing predicted and observed detrital thermochronometer ages.
This website contains documentation for Detrital MC.
If you are interested in the code itself, you can download Detrital MC from its GitHub repository at `https://github.com/HUGG/Detrital-MC/ <https://github.com/HUGG/Detrital-MC/>`_.
